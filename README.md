# GymOS Frontend

Vanilla HTML/CSS/JavaScript frontend for GymOS.

## Run

Open `index.html` directly in a browser.

The app starts in local demo mode through IndexedDB/localStorage. Backend mode can be enabled in `–Э–∞–ї–∞—И—В—Г–≤–∞–љ–љ—П` after starting the NestJS API in `D:\Features\gymos\gym-os-back`.

## Backend Mode

Default API URL is configured in `index.html`:

```js
window.FORGE_CONFIG.apiBaseUrl = "http://localhost:3000";
```

Use `–Э–∞–ї–∞—И—В—Г–≤–∞–љ–љ—П` to switch `–†–µ–ґ–Є–Љ –і–∞–љ–Є—Е` between `–Ы–Њ–Ї–∞–ї—М–љ–Є–є` and `Backend API`, then click `–Я–µ—А–µ–≤—Ц—А–Є—В–Є –њ—Ц–і–Ї–ї—О—З–µ–љ–љ—П`.
